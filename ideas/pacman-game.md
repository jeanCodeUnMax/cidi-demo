# Idée: Jeu Pac-Man en HTML/CSS/JS

## Concept
Créer un jeu Pac-Man classique jouable dans le navigateur avec HTML, CSS et JavaScript vanilla.

## Fonctionnalités Principales
- Labyrinthe classique avec murs et couloirs
- Pac-Man contrôlable au clavier (flèches)
- 4 fantômes avec IA de poursuite
- Pac-gommes à collecter (petites et grosses)
- Système de score
- Vies (3 au départ)
- Niveaux de difficulté progressive

## Contraintes Techniques
- HTML5 Canvas ou DOM pour le rendu
- CSS pour le styling de l'interface
- JavaScript vanilla (pas de framework)
- Responsive (fonctionne sur mobile)
- Performance 60 FPS

## Objectif
Générer un PRD complet avec :
- Spécifications techniques détaillées
- Architecture du code
- Devbook (guide de développement)
- Manifeste (vision et objectifs)

## Attendu
Un document PRD complet et professionnel prêt à être implémenté.
