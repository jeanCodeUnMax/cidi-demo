# Brainstorming: Application de Gestion de Tâches

## Idée
Créer une application web simple pour gérer mes tâches quotidiennes.

## Fonctionnalités Souhaitées
- Ajouter une tâche
- Marquer comme terminée
- Supprimer une tâche
- Filtrer par priorité

## Préférences
- Design minimaliste
- Sauvegarde locale (localStorage)
- Pas de backend nécessaire

## Contraintes
- Doit fonctionner offline
- Mobile-first
- Performance rapide
