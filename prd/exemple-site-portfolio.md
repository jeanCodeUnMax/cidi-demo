# PRD: Site Portfolio Personnel

## Objectif
Créer un site portfolio personnel moderne et élégant pour présenter mes compétences et projets.

> 🔄 Test de génération - DEMO (self-healing orchestrator)

## Pages Requises

### 1. Page d'Accueil
- Hero section avec nom et titre professionnel
- Photo de profil (placeholder)
- Courte bio de 2-3 phrases
- Boutons vers les réseaux sociaux (LinkedIn, GitHub, Twitter)
- Design moderne avec animation subtile

### 2. Section Compétences
- Liste de compétences techniques avec niveau (barres de progression)
- Compétences: HTML/CSS, JavaScript, Python, React, Node.js
- Design en cartes ou badges

### 3. Section Projets
- Grille de 3-4 projets
- Chaque projet: image placeholder, titre, description courte, lien
- Effet hover sur les cartes

### 4. Section Contact
- Formulaire de contact (nom, email, message)
- Bouton d'envoi stylisé
- Liens vers email et réseaux

## Design
- Palette de couleurs: tons sombres avec accent bleu/violet
- Police: moderne et lisible (sans-serif)
- Responsive: mobile-first
- Animations CSS subtiles au scroll

## Fonctionnalités
- Navigation sticky en haut
- Scroll smooth
- Mode dark par défaut
- Footer avec copyright
