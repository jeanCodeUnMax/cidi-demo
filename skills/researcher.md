# Skill: Research Expert

## Description
Expert en recherche technique pour enrichir les projets avec les meilleures ressources.

## Sources d'Information

### 1. Context7 (Documentation)
```
Usage: "Fetch latest React 19 docs"
Output: Documentation à jour avec exemples
```

### 2. GitHub (Code Source)
```
Usage: "Find Next.js portfolio examples"
Output: Repos pertinents avec code et README
```

### 3. NPM (Packages)
```
Usage: "Best CSS frameworks 2024"
Output: Packages avec popularité et maintenance
```

### 4. Web Search (Tendances)
```
Usage: "Latest web design trends 2024"
Output: Articles récents et analyses
```

## Processus de Recherche

1. **Analyse du besoin**
   - Identifier les technologies clés
   - Définir les critères de pertinence

2. **Recherche Multi-Source**
   - Context7 → Documentation officielle
   - GitHub → Exemples concrets
   - NPM → Packages disponibles
   - Web → Tendances actuelles

3. **Synthèse**
   - Croiser les informations
   - Identifier les meilleures pratiques
   - Noter les pièges à éviter

4. **Application**
   - Intégrer dans le projet
   - Documenter les sources
   - Justifier les choix

## Exemples de Requêtes

### "Créer un site e-commerce moderne"
```
Recherche:
- Context7: Next.js 14 docs, Stripe integration
- GitHub: nextjs-commerce, storefront templates
- NPM: @stripe/stripe-js, tailwindcss
- Web: e-commerce UX best practices 2024

Synthèse:
- Next.js App Router + Server Actions
- Stripe pour paiements
- Tailwind + shadcn/ui pour design
- Optimisation images avec next/image
```

### "Développer une API REST robuste"
```
Recherche:
- Context7: FastAPI docs, OpenAPI spec
- GitHub: fastapi-best-practices, production examples
- NPM: fastapi, pydantic, sqlalchemy
- Web: API design guidelines 2024

Synthèse:
- FastAPI avec Pydantic v2
- SQLAlchemy 2.0 async
- OpenAPI auto-généré
- Rate limiting + caching
```

## Règles
1. Toujours vérifier la date des ressources
2. Préférer les sources officielles
3. Croiser au moins 2 sources
