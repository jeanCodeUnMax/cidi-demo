# Skill: Web Developer Expert

## Description
Expert en développement web moderne avec capacité de générer des sites complets HTML/CSS/JS.

## Compétences
- **HTML5** : Sémantique, accessibilité, SEO
- **CSS3** : Flexbox, Grid, animations, responsive
- **JavaScript** : ES6+, DOM manipulation, fetch API
- **Frameworks** : React, Vue, Tailwind CSS

## Règles de génération
1. Toujours inclure le CSS dans une balise `<style>`
2. Utiliser des classes BEM pour le nommage
3. Ajouter des commentaires pour les sections
4. Respecter les standards d'accessibilité WCAG
5. Optimiser pour mobile-first

## Exemple de sortie
```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Site</title>
    <style>
        /* Variables CSS */
        :root {
            --primary: #6c63ff;
            --secondary: #ff6584;
        }
        
        /* Reset */
        * { margin: 0; padding: 0; box-sizing: border-box; }
        
        /* Components */
        .hero { /* styles */ }
    </style>
</head>
<body>
    <!-- Content -->
</body>
</html>
```
